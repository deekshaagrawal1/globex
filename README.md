# 🌍 GlobeX

GlobeX is a React-based interactive web application that challenges users to guess countries using clues such as region, capital, and language. It combines learning with gameplay, making geography engaging and enjoyable.

---

## ✨ Highlights

* Uses real-world country data
* Designed as an interactive quiz experience
* Clean and responsive user interface
* Focus on simplicity and user engagement

---

## 🛠️ Tech Stack

React • JavaScript • Tailwind CSS

---

## 🌐 Data Source

Country information is fetched from the REST Countries API.

---

## 🎯 Purpose

The aim of this project is to apply core React concepts like state management, effects, and component-based design while building a fun and meaningful application.
